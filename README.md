## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- (Optional) Ollama for AI features

### ⚡ One-Line Install

```bash
pip install git+https://github.com/1lo1lo1/bughunter-ai.git
🔥 First Scan (30 seconds)
# Full automated audit pipeline
bughunter mass-hunt target.com --output report.html

# Open the report
open report_target_com.html
📊 Features Overview
🔍 Discovery Layer
OSINT Sources:    crt.sh | JLDC | RapidDNS | BufferOver | Anubis
Output:           Clean subdomain list with HTTP status codes
Speed:            ~1000 subdomains/minute
🔌 Port Hunter
Scanned Ports:    21, 22, 3306, 5432, 6379, 8080, 8443, 9200
Detection:        Banner grabbing + Service fingerprinting
Smart Filter:     Removes honeypots via timing analysis
🧠 Vulnerability Engine
SSTI Detection:   {{7*7}} → validates 49 in response
LFI Patterns:     /etc/passwd, ../../, php://filter
SQLi Checks:      Error-based + Time-based detection
CORS Misconfig:   Automatic origin reflection tests
📋 Smart Filtering
Traditional scanners: 200+ false positives
BugHunter AI v3.5.0: <5 false positives (verified on snb.ch)
🛠️ Installation
Method 1: pip (Recommended)
pip install bughunter-ai
Method 2: From Source
git clone https://github.com/1lo1lo1/bughunter-ai.git
cd bughunter-ai
pip install -e .
💻 Usage Examples
🎯 Full Audit Pipeline
# Complete reconnaissance: subdomains → ports → vulnerabilities
bughunter mass-hunt bank.com --depth 3 --threads 50
🔎 Subdomain Discovery
# OSINT enumeration with live HTTP check
bughunter discover corp.com --sources crtsh,jldc --output subs.txt
🌐 Single URL Deep Scan
# Targeted vulnerability assessment
bughunter scan-url https://api.target.com/v1/users \
  --checks cors,ssti,secrets \
  --depth 2 \
  --ai
🤖 AI-Powered Analysis
# Requires Ollama running locally
ollama pull llama3.2

bughunter scan-url https://target.com --ai --model llama3.2
